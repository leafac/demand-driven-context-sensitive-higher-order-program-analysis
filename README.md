## Run Tests

Install [Node.js](https://nodejs.org/) and run:

```console
$ npm install
$ npm run build
$ npm test
```

## Build Dissertation

In addition to Node.js, install [LaTeX](https://www.latex-project.org) and [Pygments](http://pygments.org), and run:

```console
$ npm run build:dissertation
```
